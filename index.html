What is a TurboTax License Code?

A TurboTax license code is a unique 16-character code that verifies your purchase of the TurboTax software. This code is usually found on the packaging or insert of the TurboTax CD/DVD, or in the email confirmation you received after purchasing TurboTax online.

Steps to Enter Your TurboTax License Code

  1. Open your browser and go to Installturbotax.com.
  2. If you already have a TurboTax account, sign in using your credentials.
  3. If you’re a new user, click on "Create an Account" and provide the necessary details.
  4. Once logged in, locate your purchased TurboTax product and click "Download."
  5. Open the downloaded file and follow the on-screen instructions to install the software.
  6. Accept the terms and conditions to proceed.
  7. During the installation process, you’ll be prompted to enter your 16-character license code.
  8. Type in the code and click "Continue."
  9. Once the code is verified, your TurboTax software will be activated and ready to use.
